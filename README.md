## Quick Start

Clone the Repository:

```bash
git clone https://github.com/mahankals/ExpressBooster-app.git myApp
cd myApp
```

Install dependencies:

```bash
npm install
```

Start your Express app

```bash
npm start
```

Browse your app at [http://localhost:3000/](http://localhost:3000/)
